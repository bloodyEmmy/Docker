Сетевой драйвер ([[Сетевые драйверы Docker]]) соединяет несколько [[Контейнер]] по [[Сеть хоста]]. Позволяет построить изолированную сеть overlay или оверлейную сеть поверх существующей сети на хосте.
	При установке [[Docker]] оверлейная сеть не создается по умолчанию, для этого нужно инициализировать [[Docker Swarm]] или присоединить узел ([[Узлы в Swarm (Nodes)]]) к Docker Swarm. Тогда создается не только [[Ingress Network]], но и [[Docker_gwbridge]].
